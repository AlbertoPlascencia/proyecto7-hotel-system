# Sistema Web Hotelero

Proyecto desarrollado para la materia **Proyecto VII**.

## Sprint 1 – Estructura inicial del sistema

En este sprint se establecieron las bases del proyecto:

- Creación del repositorio en GitHub
- Definición de la estructura del sistema
- Implementación de un frontend básico
- Configuración inicial del backend
- Definición preliminar de la base de datos

## Sprint 2 – Gestión de reservaciones

En este sprint se implementó:

- módulo de gestión de habitaciones
- sistema de reservaciones
- validación básica de disponibilidad
- flujo operativo del sistema hotelero
