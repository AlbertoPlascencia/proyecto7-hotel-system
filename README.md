# Sistema Web Hotelero

Proyecto desarrollado para Proyecto VII

Este proyecto simula un sistema básico de gestión hotelera que permite administrar habitaciones y registrar reservaciones de clientes. El desarrollo del sistema se organiza utilizando la metodología Scrum, dividiendo el trabajo en diferentes sprints para implementar y mejorar progresivamente las funcionalidades.



# Estructura del proyecto

El sistema está organizado en tres componentes principales:

backend → lógica del servidor y simulación del sistema  
frontend → interfaz del sistema en el navegador  
database → estructura de la base de datos  



# Sprint 1 – Estructura inicial del sistema

En este sprint se establecieron las bases del proyecto:

- Creación del repositorio en GitHub
- Definición de la estructura inicial del sistema
- Implementación de un frontend básico
- Configuración inicial del backend
- Definición preliminar de la base de datos

Este sprint permitió crear la base necesaria para comenzar a desarrollar las funcionalidades del sistema.



# Sprint 2 – Gestión de reservaciones

En este sprint se implementaron las funcionalidades principales del sistema hotelero:

- Módulo de gestión de habitaciones
- Sistema de registro de reservaciones
- Asociación de clientes con habitaciones
- Manejo básico de fechas de entrada y salida
- Flujo inicial del sistema de reservaciones

Con estas funcionalidades el sistema comenzó a simular operaciones básicas dentro de un entorno hotelero.



# Sprint 3 – Mejora del sistema y validaciones

En el tercer sprint se realizaron mejoras en la lógica del sistema y en la interacción del usuario con la interfaz.

Entre las mejoras implementadas se encuentran:

- Validación de disponibilidad de habitaciones
- Mejora en el flujo de registro de reservaciones
- Visualización de reservaciones registradas
- Mejora en la navegación entre módulos del sistema
- Simulación de cambios de estado de habitaciones

Estas mejoras permiten que el sistema evolucione hacia un funcionamiento más cercano al de un sistema hotelero real.


# Funcionalidades actuales del sistema

Actualmente el sistema permite:

- Iniciar sesión en el sistema
- Visualizar habitaciones disponibles
- Cambiar el estado de habitaciones
- Registrar reservaciones
- Visualizar reservaciones registradas
